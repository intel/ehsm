## eHSM-Client SDK for rust

This repo contains the ehsm client sdk for rust applications.

It will provide the following interfaces:

---
**Key Management APIs:**
- create_key
- encrypt
- decrypt
- sign
- verify
- asymmetric_encrypt
- asymmetric_decrypt
- generate_dataKey
- generate_datakey_without_plaintext
- export_dataKey
- get_publickey

- GetVersion
- Enroll
- ListKey
- DeleteKey
- DeleteALLKey
- EnableKey
- DisableKey
---

**Secret Management APIs:**
- CreateSecret
- UpdateSecretDesc
- PutSecretValue
- ListSecretVersionIds
- ListSecrets
- DescribeSecret
- DeleteSecret
- GetSecretValue
- RestoreSecret
